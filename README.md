# `l3gd20`

> A platform agnostic driver to interface with the L3GD20 and L3GD20H (gyroscope) which supports
> both the I2C and SPI interface.

## Examples

You should find at least one example in the [f3] crate.

[f3]: https://docs.rs/f3/~0.5
